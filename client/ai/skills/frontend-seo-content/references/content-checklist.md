# Content Checklist

Use this file when the task touches public-page copy together with SEO.

## Check

- Does the page have one clear primary heading?
- Does the title match the visible intent of the page?
- Does the description summarize the page in plain language?
- Is there enough visible content to justify indexation?
- Is the route better handled as `noindex` because it is thin, duplicate, or user-private?

## For Dynamic Public Pages

- Keep title templates predictable.
- Avoid placeholder descriptions.
- Prefer real page entities such as username, product title, event title, or blog title.
