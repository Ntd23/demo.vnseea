# Page Review Checklist

- Confirm the route wrapper is still thin.
- Confirm the real page lives in `src/<context>/presentation/pages/*`.
- Check spacing against nearby pages in the same flow.
- Check labels, body text, and section titles for typography drift.
- Check border radius, borders, and shadows for consistency.
- Check active, hover, and disabled states.
- Check mobile and desktop layout.
- Check that the page is not hiding a proxy or asset problem with local hacks.
